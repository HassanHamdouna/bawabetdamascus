# bawabetdamascus

BawabetDamascus is a comprehensive internal management Flutter app for restaurants.  It allows waiters to take orders, send them to the kitchen, and print invoices.  Chefs can track and update order status. Accountants can manage payments and track tables.  System managers can add employees, manage menu items, and oversee daily operations.  Owners and partners can view reports, finances, and monitor all activities.  Supports multi-language (Arabic, French, English) and dark/light mode.  ## Getting Started  This project is a starting point for a Flutter application.
